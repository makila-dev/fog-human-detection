🌫️ About Our System

The Fog-Based Human Detection System is an AI-powered surveillance solution designed to detect humans in foggy, smoky, and low-visibility environments.

Traditional surveillance systems struggle in fog because visibility becomes poor. Our system uses advanced image enhancement and deep learning to identify human presence accurately.

Using YOLOv8 deep learning technology, the system performs real-time detection with high speed and accuracy.

🚀 Technologies Used

  • 🐍 Python Programming
  • 🌐 Flask Web Framework
  • 📸 OpenCV Image Processing
  • 🤖 YOLOv8 Object Detection
  • 🧠 Artificial Intelligence
  • 🎥 Real-Time Camera Monitoring
